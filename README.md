# 📝 To-Do App

A simple and responsive To-Do List application built with **HTML, CSS, and JavaScript** that helps users manage daily tasks efficiently.

Users can add tasks, mark them as completed, delete tasks, and enjoy persistent storage using the browser's **Local Storage**, ensuring tasks remain available even after refreshing or reopening the browser.

## ⚙️ How It Works

### Add a Task

1. Enter a task in the input field.
2. Press Enter or submit the form.
3. The task is added to the list and saved to Local Storage.

### Complete a Task

1. Click the checkbox beside a task.
2. The completion status is updated instantly.
3. Changes are automatically saved.

### Delete a Task

1. Click the delete icon.
2. The task is removed from the list.
3. Local Storage is updated accordingly.

### Persistent Storage

The application uses the browser's Local Storage to save all tasks, allowing users to retain their data even after refreshing or closing the browser.
